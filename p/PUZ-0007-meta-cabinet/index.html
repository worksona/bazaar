How the meta works

Each fragment below was enciphered with one of the cabinet’s five techniques, using the very key you recovered when you solved that rung. Type the right key into each lock — the keyword you cracked, or the rotor settings you searched out — and the fragment decodes to a single word. The five words, in order, are the cabinet’s final secret. Get all five and the flag seals itself.

Don’t remember a key? Each lock links back to its rung, and there’s a spoiler at the very bottom — but try from memory first.
