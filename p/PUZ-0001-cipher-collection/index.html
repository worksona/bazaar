1

Learn — what it is & why it mattered

Two minutes of reading. Everything below uses these three ideas.

A Vigenère cipher is just a stack of Caesar shifts. You choose a secret keyword. Write it under your message, repeating it, letter for letter. Each message letter is pushed forward through the alphabet by the keyword letter beneath it (A = no shift, B = +1, C = +2 …). Because the same plaintext letter is enciphered differently depending on where it lands, the tell-tale letter frequencies that give away a simple cipher get smeared away.

Why it was used

Assembled by Bellaso in 1553 and later named for Blaise de Vigenère, it was reckoned unbreakable. As late as 1917, Scientific American still called it “impossible of translation.”

The curious turn — how it falls

The keyword repeats, and repetition leaves a rhythm. Charles Babbage and later Friedrich Kasiski noticed that the cipher is really n ordinary Caesar ciphers interleaved — one per keyword letter. Find n (the key length), split the message into n columns, and each column is a plain Caesar shift you can read off by frequency. That two-step attack is exactly what Station 3 lets you run by hand.

Best use today: share a keyword in person — a street you grew up on, a pet’s name — and you can hide one line in a postcard that a passing reader sees as nonsense. The keyword never travels with the message. That’s the whole trick.

Analyze — the attacker’s workbench

No keyword? Recover it from the ciphertext alone, in two moves.

Move A · find the key length

English letters clump together (lots of E, T, A; few Z, Q). If we slice the text into the right number of columns, each column is plain English in disguise and stays “clumpy.” The wrong number scrambles them flat. The index of coincidence just measures that clumpiness — higher means more English-like.

Move B · recover the keyword, column by column

Each column is one Caesar shift. We try all 26 possible shifts and keep the one whose letters land closest to English’s natural fingerprint (lots of E, T, A; almost no Q, Z). That best-fit shift is the keyword letter for that column. The tall bar below is the column’s most common letter — usually a disguised E.
