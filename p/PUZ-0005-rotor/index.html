1

Learn — a new alphabet every key

Three wheels of scrambled wiring, stepping like an odometer, plus a mirror.

A rotor machine sends each letter through three wired wheels, bounces it off a reflector, and back out through the wheels again. After every keystroke the right wheel clicks forward one step — and when it completes a turn it nudges the next — so the substitution changes with every letter. This is the Vigenère idea made mechanical, with a key period of thousands. The reflector also makes it self-reversing: the same settings that encode a message decode it.

Its strength — and its fatal gift

The wiring of the wheels was public; the secret was only which order they sat in and where each started — here, 3 wheels in any of 6 orders × 26×26×26 start positions ≈ 105,000 settings. Vast for a human, nothing for a search. And the reflector left a crack: a letter can never encrypt to itself. Try every setting, score each decrypt for how English it looks, and the true one stands out like a lit window. Station 3 runs exactly that search — the modern machine that broke the wartime one, in miniature.

Analyze — the settings search

You don’t know the order or the start letters. So try all 105,456 of them.

The search decrypts the message under every wheel order and start position, scoring each result by how closely its letter mix matches English. Only the true settings yield real words, so the top score is almost always the answer.
