Learn — same letters, new order

Write the message into a grid by rows; read it back out by columns.

A columnar transposition writes your message across a grid whose width is set by a keyword, then reads the grid down its columns — taking the columns in the alphabetical order of the keyword’s letters. With keyword QUAY the columns are read A, Q, U, Y → 3rd, 1st, 2nd, 4th. The letters are all still there; only their order has changed.

Q U A Y M E E T M E B Y T H E W A L L X → read columns A,Q,U,Y → EBELMMTAEEHL TYWX

Why frequency analysis fails — and what works instead

Because every original letter survives untouched, the ciphertext has the exact letter counts of English. Counting tells you nothing. The way in is the shape: guess the number of columns, cut the ciphertext back into those columns, then shuffle the columns until the rows read as language. It’s a jigsaw, not a tally.
